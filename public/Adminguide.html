홈페이지 관리 가이드 관리자용

관리자 페이지에서 바로 확인·복사 가능한 HTML 버전입니다. (접속 경로: /admin)

1. 포트폴리오 (행사 사진) 업로드
Google Drive에 업로드하면 홈페이지에 자동 반영됩니다.
경로: Google Drive > 플래너스컴퍼니행사사진 폴더 사진을 폴더에 업로드 후 설명(alt/설명 텍스트) 추가하기: 1) Google Drive 폴더에 업로드한 사진 선택 2) 오른쪽 상단 점 3개 클릭 → 파일 정보 3) 세부정보 탭에서 '설명' 입력 → 저장 설명 입력 시: 사진 하단에 해당 텍스트가 노출됩니다.
2. 문의사항 접수 관리
방문자가 문의를 작성하면 Google Sheets에 자동 저장되고 이메일 알림이 발송됩니다.
플로우: 1) 방문자: 홈페이지 문의하기폼 작성 2) 시스템: Google Sheets로 데이터 전송 3) 시스템: 관리자 이메일로 알림 발송 확인 방법: - Google Sheets: 최신 행 확인 - 이메일: 수신함의 알림 확인
3. 공지사항 관리 (Admin)
관리자 페이지에서 공지 등록·수정·삭제가 가능합니다.
접속 주소: /admin 사용 방법: 1) 로그인 → 공지사항 관리 화면 접속 2) 등록: 제목 / 내용 입력 → 저장 3) 수정: 기존 공지의 수정버튼 클릭 → 내용 수정 → 저장버튼 클릭 4) 삭제: 삭제 버튼 클릭 → 확인 추가 기능: - 최근 게시물이 상단 정렬 - 이미지는 링크(URL)로 본문에 삽입 가능 - 공지사항과 함께 파일 첨부: 1) 구글 클라우드 공지사항 파일 업로드 폴더에 파일 업로드 2) 업로드한 파일을 선택하고 링크(URL)를 복사를 누름 3) 공지사항 작성 시, URL 입력란에 붙여넣기 → 게시물에 사진/파일과 함께 공지 등록 가능
